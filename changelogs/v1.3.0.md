## 📌 Version 1.3.0 – Bug Fixes & Stability Release
### 🐛 Fixed
#### Critical Bugs
- **Null Reference Exception** (`PermissionHandler.php:28`)
  - Fixed crash when `getGroup()` returns `null`
  - Implemented automatic fallback to "guest" group
- **Non-existent Group** (`PermissionHandler.php:29`)
  - Fixed crash when a user had a non-existent group
  - Now verifies group existence before applying permissions
- **Users Not Loaded from File** (`UserManager.php:29-34`)
  - Fixed bug where users connecting after server startup didn't load saved data
  - Implemented `loadUserFromFile()` method for dynamic loading
- **Incorrect Default Group** (`UserManager.php:23`, `EventListener.php:24`)
  - Fixed default group from "default" to "guest"
  - Updated verification logic with null-coalescing operator
- **Wrong Group in GroupDel** (`GroupDel.php:37`)
  - Fixed bug where when a group was deleted, players were assigned to "default" (non-existent)
  - Now correctly moved to "guest"
#### Memory & Performance
- **Memory Leak** (`PermissionHandler.php`)
  - Fixed memory leak on permission attachments when players left the server abnormally
  - Implemented automatic cleanup with `onQuit()` handler
- **Permissions Not Reapplied on Reconnect** (`EventListener.php`)
  - Improved player reconnection handling
#### Code Quality
- **Unused Import** (`GroupRmPerm.php:8`)
  - Removed unused `SubCommand` import
### 📝 Updated
- Improved overall plugin stability
- Optimized user and group data loading
