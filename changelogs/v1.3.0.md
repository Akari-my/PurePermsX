## 📌 Version 1.3.0 – Bug Fixes & Stability Release

### 🐛 Fixed

#### Critical Bugs
- **Null Reference Exception** (`PermissionHandler.php:28`)
  - Risolto il crash quando `getGroup()` ritorna `null`
  - Implementato fallback automatico al gruppo "guest"

- **Gruppo inesistente** (`PermissionHandler.php:29`)
  - Risolto il crash quando un utente aveva un gruppo non esistente
  - Ora verifica l'esistenza del gruppo prima di applicare i permessi

- **Utenti non caricati dal file** (`UserManager.php:29-34`)
  - Risolto il bug per cui gli utenti che si connettevano dopo l'avvio del server non caricavano i dati salvati
  - Implementato metodo `loadUserFromFile()` per caricamento dinamico

- **Gruppo di default errato** (`UserManager.php:23`, `EventListener.php:24`)
  - Corretto il gruppo di default da "default" a "guest"
  - Sistemata la logica di verifica con null-coalescing operator

- **Gruppo errato in GroupDel** (`GroupDel.php:37`)
  - Risolto il bug per cui quando si eliminava un gruppo, i giocatori venivano assegnati a "default" (inesistente)
  - Ora vengono spostati correttamente a "guest"

#### Memory & Performance
- **Memory Leak** (`PermissionHandler.php`)
  - Risolto il leak di memoria sugli allegati dei permessi quando i giocatori abbandonavano il server in modo anomalo
  - Implementato cleanup automatico con `onQuit()` handler

- **Permissions non riapplicate al reconnect** (`EventListener.php`)
  - Migliorata la gestione della riconnessione dei giocatori

#### Code Quality
- **Import non utilizzato** (`GroupRmPerm.php:8`)
  - Rimosso import `SubCommand` non utilizzato

### 📝 Updated
- Migliorata la stabilità generale del plugin
- ottimizzato il caricamento dei dati utente e gruppi
